# Lawyer Management System

A Django-based web application developed as part of university coursework.

## Features

- User authentication (Login / Logout / Register)
- Role-based dashboards (Client & Lawyer)
- Case management
- Appointment booking system
- Automatic time slot generation
- Reports:
  - Completed consultations (date range)
  - Revenue per lawyer
  - Appointment distribution by speciality and lawyer

## Tech Stack

- Python 3
- Django
- SQLite
- HTML (Django Templates)
